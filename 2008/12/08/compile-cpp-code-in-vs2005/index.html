在vs2005中编译c++代码！

发表于 2008-12-08 更新于 2016-09-04 分类于 日积月累 Disqus：

在vs2005里面不能够像以前vc6.0里面那样直接编译cpp文件了。
你首先要建立一个空项目，然后把cpp文件添加到源文件里面，那样就可以编译执行了。
具体步骤：
文件->新建->项目
在出来的窗口中选择：常规->空项目->文件名（自己编辑）->位置->确定。
创建了空项目之后把你要编译的cpp文件添加到源文件（右键点击源文件添加就行了）里面，按F5是调试模式，ctrl+F5是执行模式（不调试）。
添加方法：
右键源文件->添加
再选“现有项”或“新建项”
还有，vs2005里面写c++引入头iostream文件不能够写#include<iostream.h>，而要写成
#include
using namespace std;
也就是申明引用命名空间
而且不支持默认 int所以在main函数前面要写上int，就是int main().
good lucky
