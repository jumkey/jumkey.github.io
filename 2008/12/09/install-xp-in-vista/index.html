教你在Vista下安装XP

发表于 2008-12-09   |   分类于 日积月累   |  

不知有多少人使用vista，也不知道我写这篇文章有没有看。（无奈）
在笔记本论坛（本人常泡的论坛）上看到的朋友基本都是用vista的，也有人是双系统的，xp的人也不算少。
今天我介绍的是在高版本系统下装低版本系统，也就是在vista下装xp。
前期准备：
一。需要工具：1.Microsoft .NET Framework 2.0（很多软件都需要这个支持） 2.VistaBootPRO_3.x（调出启动菜单） 3.xp光盘 4.上网拨号工具（家里基本用不到）。
二。准备驱动：如果是品牌机，那么驱动可以去官方网站下载（要xp的驱动哦），组装机就去下载万能驱动吧（不知道有没有这个东西，瞎编的）。
三。一颗大脑。
工作开始：
把xp光盘放入光驱，设置cd启动，一般按delete键，也可能是F2.自己试试，哪个行就那个。
然后出现Press any key……，就是叫你按任意键继续。然后就开始安装了。
安装完成后，关键到了，先安装工具中的Microsoft .NET Framework 2.0，在安装VistaBootPRO_3.x。

  1. 安装完VistaBootPRO_3.2.0启动后界面如下

    开始界面是要你备份，先点击我用黄色涂过的按钮，指定备份路径。ok
  2. 点击涂成黄色的按钮如下图
  3. 选中黄色选项，再点黄色按钮如下图
  4. 点击黄色按钮，选中Run Diagn0stics或者直接按Alt+R。如下图
  5. 出现如下界面
  6. 如果方框内不是乱码，可以直接点黄色按钮，也可以选中其中一个，选中黄色的在方框内填写自己喜欢的启动名字

over！
ok了
等你下次启动就看到了两个启动菜单了，随便你选择！爽吧
