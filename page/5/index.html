解决了一个WordPress后台设置的乱码问题

发表于 2008-12-31 更新于 2016-09-04 分类于 日积月累 Disqus：

自从升级了官方的中文版本后，一直有一个问题困扰这我，就是在后台设置的时候会出现乱码。

比如：当我某个设置成功后，会出现一个跳转页面，这个页面就出现乱码了，每次设置都会出现这个跳转页面，而且是乱码的，看着都心烦。

后来分析了一下，那个页面是admin-footer.php控制的，原因貌似是UTF-8和gb2312两个编码的问题，其实只要出现乱码问题90%就是这两个编码的导致的。

本来admin-footer.php和admin-header.php是结合使用的，在admin-header.php上有指定文件的编码格式语句，所以后台首页上admin-footer.php不会显示乱码，

但是当admin-footer.php单独使用是就出现问题了，乱码出现了。那个跳转页面就是单独使用admin-footer.php的。

解决方法：人工指定编码格式，用记事本打开admin-footer.php然后另存为UTF-8编码格式，再上传覆盖ftp的文件就ok了。

解决了这个问题心情变得轻松了很多。

提交网站到百度，谷歌，雅虎，搜狗等各个搜索引擎

发表于 2008-12-29 更新于 2016-09-08 分类于 杂七杂八 Disqus：

一个新站建立好以后，即使是像我这样的个人博客小站，我觉得首先并且有必要做的一件事就是提交网站地址到各大小搜索引擎。如果是商业运行或者稍大一点的站点如果要想获得流量和浏览量更是应该要做这件事，况且比起其它网站建设工作来说，提交网站到搜索引擎实在是花很小的时间和精力的一件事了。
以下是从网络上搜集到的部分免费提交网站搜索引擎的入口地址：

1.Google(谷歌)-这个就不用说了


收录入口：http://www.google.com/addurl/?hl=zh-CN
收录时间：很快，几天就收录了

2.百度-也不用说了，中文领域最大最有名的搜索引擎


收录入口：http://www.baidu.com/search/url_submit.html
收录时间：我的还没收录，鄙视之
3.雅虎中国-现已归阿里巴巴旗下


收录入口：http://search.help.cn.yahoo.com/h4_4.html
收录时间：数周至数月
4.微软Live Search-据说与国内114搜索同数据库，前一阵子好象关闭了高级搜索功能，但现在又恢复了


收录入口：http://search.live.com/docs/submit.aspx
收录时间：数周

5.Yahoo! Search-英文Yahoo搜索,除可提交website url，还可提交Site Feed.


收录入口：https://siteexplorer.search.yahoo.com/submit/
收录时间：数周
6.搜狐/搜狗

收录入口：已停用，说是Google搜录之后自动收录。（假的）

收录入口：http://iask.com/guest/add_url.php
收录时间：一个月内
8.中搜－中国搜索？


收录入口：http://ads.zhongsou.com/register/page.jsp
收录时间：一个月内
