教你在Vista下安装XP

发表于 2008-12-09 更新于 2018-11-12 分类于 日积月累 Disqus：

不知有多少人使用vista，也不知道我写这篇文章有没有看。（无奈）
在笔记本论坛（本人常泡的论坛）上看到的朋友基本都是用vista的，也有人是双系统的，xp的人也不算少。
今天我介绍的是在高版本系统下装低版本系统，也就是在vista下装xp。
前期准备：
一。需要工具：1.Microsoft .NET Framework 2.0（很多软件都需要这个支持） 2.VistaBootPRO_3.x（调出启动菜单） 3.xp光盘 4.上网拨号工具（家里基本用不到）。
二。准备驱动：如果是品牌机，那么驱动可以去官方网站下载（要xp的驱动哦），组装机就去下载万能驱动吧（不知道有没有这个东西，瞎编的）。
三。一颗大脑。
工作开始：
把xp光盘放入光驱，设置cd启动，一般按delete键，也可能是F2.自己试试，哪个行就那个。
然后出现Press any key……，就是叫你按任意键继续。然后就开始安装了。
安装完成后，关键到了，先安装工具中的Microsoft .NET Framework 2.0，在安装VistaBootPRO_3.x。

  3. 选中黄色选项，再点黄色按钮如下图
  4. 点击黄色按钮，选中Run Diagn0stics或者直接按Alt+R。如下图

在vs2005中编译c++代码！

发表于 2008-12-08 更新于 2016-09-04 分类于 日积月累 Disqus：

在vs2005里面不能够像以前vc6.0里面那样直接编译cpp文件了。
你首先要建立一个空项目，然后把cpp文件添加到源文件里面，那样就可以编译执行了。
具体步骤：
文件->新建->项目
在出来的窗口中选择：常规->空项目->文件名（自己编辑）->位置->确定。
创建了空项目之后把你要编译的cpp文件添加到源文件（右键点击源文件添加就行了）里面，按F5是调试模式，ctrl+F5是执行模式（不调试）。
添加方法：
右键源文件->添加
再选“现有项”或“新建项”
还有，vs2005里面写c++引入头iostream文件不能够写#include<iostream.h>，而要写成
#include
using namespace std;
也就是申明引用命名空间
而且不支持默认 int所以在main函数前面要写上int，就是int main().
good lucky

解决VISTA多重网络不能上网的方法

发表于 2008-12-08 更新于 2016-09-04 分类于 日积月累 Disqus：

方法一（本人没有测试）：连接后vista显示多重网络，要求我断开一个，但是我禁用的话两个网络就都消失了，不断开的话又不能上网，请问着怎么解决啊
其实很简单，把网关回复到以前的状态就行了，在vista下输入命令 cmd 接下来就输入 ipconfig/release 或者 ipconfig/renew 肯定能解决问题的～～～

或者修改连接的高级默认网关勾掉去掉就OK了。

方法二（本人已测试成功）：部分VISTA装了ps cs3后，网络连接会多了个未识别网络，导致无法上网，实际上是影响到本机的路由表，多了一条奇怪的路由。之前试过写一个批处理，开机自动运行，用route delete 来删除这条路由，但实在不是治根之法，后来终于知道是ps cs3添加的一个服务搞鬼，只要禁用它就行。
原文：http://ray-bluelife.blogbus.com/logs/10079668.html
最近发现多出一个进程mDNSResponder.exe，百度一下才知道是安装PhotoShop CS3所带的，而且还会多出一项服务：

再找：Bonjour，原名Rendezvous，是苹果电脑公司在其开发的操作系统Mac OS X10.2版本之后引入的服务器搜索协议所使用的一个商标名。适用于LAN，Bonjour使用多点传送域名系统服务记录来定位各种设备，比如打印机或者其他计算机，以及另外设备上的服务。

解决vista下windows media player11外挂字幕不能播放视频问题

发表于 2008-12-07 更新于 2018-11-12 分类于 日积月累 Disqus：

问题描述：下载了avi的视频，用wmp11可以打开，但是在同一个文件夹下放一个与avi同名字幕文件，视频就不能播放。
解决办法：下载一个最新版的Vobsub（VSFilter）【就一个VSFilter.dll文件】 下载地址：VSFilter_20080828.zip
安装Vobsub：把下载的VSFilter.dll文件放到system32文件夹下，然后在运行里输入”regsvr32 vsfilter.dll”注册文件
在运行里输入”rundll32 vsfilter,DirectVobSub”，就可以进入字幕插件的配置面板。
安装好之后打开wmp11调出经典菜单，方法：右键标题栏下方黑色区域->选显示经典菜单。
在”播放”选项里选择”字幕”开启字幕
这样基本解决问题了。
问题原因：估计是wmp11没有安装字幕插件或者版本太低！
